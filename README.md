# Survival Arena（生存竞技场）

多人竞技 PVE FPS，基于 Unreal Engine 5.7 开发。

## 背景

击杀怪物积累积分以解锁更强武器。**第一个达到 10 分的人活着离开**。

## 游戏规则

- **胜利条件**：个人分数 ≥ 10 分
- **失败条件**：场上存活玩家 = 0
- **死亡惩罚**：分数归零，武器等级重置，5 秒后重生
- **得分方式**：击杀 NPC → 最后一击者 +1 分
- **子弹升级**：

| 等级 | 条件 | 效果 |
|------|------|------|
| Tier 0 | < 3 分 | 基础子弹，伤害 25 |
| Tier 1 | ≥ 3 分 | 伤害 40，子弹 1.2x |
| Tier 2 | ≥ 6 分 | 伤害 60，子弹 1

## 操作方式

| 按键 | 操作 |
|------|------|
| W / A / S / D | 移动 |
| 鼠标移动 | 视角控制 |
| 鼠标左键 | 射击 |
| 空格键 | 跳跃 |

## 技术实现

### 引擎与技术栈

- **Unreal Engine 5.7**
- **C++ 为主**，蓝图仅用于 UI、动画蓝图、数据资产
- **Enhanced Input** 处理玩家输入
- **StateTree** 模板 AI（NPC 改用自定义 C++ 追逐逻辑）
- **UMG** 实现 UI 控件

### 网络架构

采用 **服务器权威**（Server Authority）模式：

- 所有伤害、计分、刷怪逻辑在 Server 端执行
- `Replicated` 变量自动同步到所有客户端
- `Server RPC` 处理客户端→服务器的安全调用
- `NetMulticast RPC` 处理服务器→所有客户端的通知
